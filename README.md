# Overwiew

This WordPress plugin allow you to purge CloudFlare cache using CloudFlare API.

## Prerequisites**
** for 1.0 version
- cURL
- WordPress (5.8.2 tests pass)

Example of cURL installation on Debian:
```bash
apt install curl
```
